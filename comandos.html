Comandos Básicos - Linux Ubuntu

Lista de comandos adquiridas no curso “Linux I: Conhecendo e utilizando o terminal”. Este curso faz parte da Formação “Administradores de Redes”, ministrado pela alura.

  1. comando --help - exibe informações sobre o comando
  2. pwd - informa em que diretório está
  3. ls - lista os arquivos e diretórios, dentro do diretório atual.
  4. echo - imprime na tela
  5. echo “texto” > nomeDoArquivo.txt cria o arquivo nomeDoArquivo.txt que contém a palavra texto.
  6. echo “texto novo” >> nomeDoArquivo.txt acrescenta a palavra "novo" no arquivo nomeDoArquivo.txt
  7. cat - recebe o nome do arquiv como argumento e imprime seu conteúdo, exemplo: cat lista.txt
  8. cat *.txtdescreve todos os arquios .txt
  9. head nomeDoArquivo.txt - exibe as dez primeiras linhas do arquivo
  10. head -n 3 nomeDoArquivo.txt - exibe as 3 primeiras linhas do arquivo
  11. tail nomeDoArquivo.txt - exibe as dez últimas linhas do arquivo, também aceita o argumento -n
  12. less nomeDoArquivo.txt - abre o arquivo no terminal, podendo ser fechado com a tecla q
  13. touch nomeDoArquivo.txt - toca no arquivo, alterando a data de edição
  14. vi nomeDoArquivo.txt - permite editar o arquivo direto do terminal

    15. tecla i para inserir texto na posição atual;


    tecla a para inserir texto no caracter seguinte;


    tecla A para inserir texto no final da linha;


    tecla x para remover um caracter, pode receber um número antes do x para deletar mais de um caractar por vez;


    tecla Esc para sair do modo de edição;


    tecla G move o cursor para a última linha;


    teclas 0G1G move o cursor para a primeira linha;


    teclas 7G move o cursor para a sétima linha (7 representa qualquer número);


    teclas dd para remover uma linha inteira;


    teclas yy para copiar uma linha inteira;


    teclas 9yy para copiar nove linhas (9 representa qualquer número);


    tecla p para colar;


    teclas 4p para colar quatro vezes o mesmo conteúdo (4 representa qualquer número);


    utilizar as teclas de seta para navegar pelo documento;


    tecla : para habilitar linha de comando;


    comando w para salvar (w=write);


    comando q para sair (q=quit);


    comando q! para sair sem salvar;


    comando wq para sair e salvar;


    tecla /nomeDaPalvra realiza uma busca por nomeDaPalvra (n vai para a próxima ocorrencia; N volta para a ocorrencia anterior);


  15. clear - Limpa a tela do terminal.(atalho CTRL+L)
  16. man - Exibe o manual do comando, exmplo: man ls - exibe o manual do ls
  17. cd - muda de diretório (change directory), exemplo: cd /c/documentos, porém, cd sem argumentos nos leva para o dirtório base
  18. ls / - visualiza o diretório raiz
  19. mkdir - cria novo diretório, recebe o nome do diretório exemplo: mkdrir pasta
  20. rm arquivo.txt - remove o arquivo com nome arquivo.txt
  21. rmdir - remove diretório vazio
  22. rm -r - para remover diretório com algum conteúdo
  23. cp - copia o conteúdo de um arquivo em outro, obs.: para cp copiar um diretório, com conteúdo, é preciso utilizar o argumento -r
  24. mv - move o arquivo ex.: mv arq.txt novoDiretorio
  25. zip nomeArq.zip nomeDir - compacta o diretório nomeDir, criando um arquivo nomeArq.zip
  26. zip -r nomeArq.zip nomeDir - utiliza-se o -r para compactar tudo que está dentro do diretório
  27. zip -rq nomeArq.zip nomeDir - o argumento q resume as informações na tela
  28. unzip - descompacta os arquivos
  29. unzip -l - lista o conteúdo do arquivo
  30. tar -czf nomeDoArquivo.tar.gz nomeDoDiretorio - compacta o diretorio gerando um arquivo .tar.gz

    31. c - cria; z - zipa; f - file, ou seja, nome do arquivo

  31. tar -xzf nomeDoArquivo.tar.gz - para descompactar
  32. tar -vxzf nomeDoArquivo.tar.gz - o argumento v, mostra detalhes do processo
  33. tar -cjf nomeDoArquivo.tar.bz2 nomeDoDiretorio - forma mais eficiente de compactar, pode levar mais tempo
