Comandos Básicos

Lista de comandos adquiridas no curso “Linux I: Conhecendo e utilizando o terminal”. Este curso faz parte da Formação “Administradores de Redes”, ministrado pela alura.

  1. pwd - informa em que diretório está
  2. ls - lista os arquivos e diretórios, dentro do diretório atual.
  3. echo - imprime na tela
  4. echo “texto” > nomeDoArquivo.txt cria o arquivo nomeDoArquivo.txt que contém a palavra texto.
  5. echo “texto novo” >> nomeDoArquivo.txt acrescenta a palavra "novo" no arquivo nomeDoArquivo.txt
  6. cat - recebe o nome do arquiv como argumento e imprime seu conteúdo, exemplo: cat lista.txt
  7. cat *.txtdescreve todos os arquios .txt
  8. clear - Limpa a tela do terminal.(atalho CTRL+L)
  9. man - Exibe o manual do comando, exmplo: man ls - exibe o manual do ls
  10. cd - muda de diretório (change directory), exemplo: cd /c/documentos, porém, cd sem argumentos nos leva para o dirtório base
  11. ls / - visualiza o diretório raiz
  12. mkdir - cria novo diretório, recebe o nome do diretório exemplo: mkdrir pasta
  13. rmdir - remove diretório vazio
  14. rm -r - para remover diretório com algum conteúdo
