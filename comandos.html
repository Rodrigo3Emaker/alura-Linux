Comandos Básicos - Linux Ubuntu

Lista de comandos adquiridas no curso “Linux I: Conhecendo e utilizando o terminal”. Este curso faz parte da Formação “Administradores de Redes”, ministrado pela alura.

Arquivos e Diretórios

  1. nomeDoComando --help - exibe informações sobre o comando, exemplo: ls --help
  2. pwd - informa em que diretório está
  3. ls - lista os arquivos e diretórios, dentro do diretório atual.
  4. echo - imprime na tela
  5. echo “texto” > - nomeDoArquivo.txt - cria o arquivo nomeDoArquivo.txt que contém a palavra texto.
  6. echo “texto novo” >> nomeDoArquivo.txt - acrescenta a palavra "novo" no arquivo nomeDoArquivo.txt
  7. cat - recebe o nome do arquiv como argumento e imprime seu conteúdo, exemplo: cat lista.txt
  8. cat *.txt - descreve todos os arquios .txt
  9. head nomeDoArquivo.txt - exibe as dez primeiras linhas do arquivo
  10. head -n 3 nomeDoArquivo.txt - exibe as 3 primeiras linhas do arquivo
  11. tail nomeDoArquivo.txt - exibe as dez últimas linhas do arquivo, também aceita o argumento -n
  12. less nomeDoArquivo.txt - abre o arquivo no terminal, podendo ser fechado com a tecla q
  13. touch nomeDoArquivo.txt - toca no arquivo, alterando a data de edição
  14. clear - Limpa a tela do terminal.(atalho CTRL+L)
  15. man - Exibe o manual do comando, exmplo: man ls - exibe o manual do ls
  16. cd - muda de diretório (change directory), exemplo: cd /c/documentos, porém, cd sem argumentos nos leva para o dirtório base
  17. ls / - visualiza o diretório raiz
  18. mkdir - cria novo diretório, recebe o nome do diretório exemplo: mkdrir pasta
  19. rm arquivo.txt - remove o arquivo com nome arquivo.txt
  20. rmdir - remove diretório vazio
  21. rm -r - para remover diretório com algum conteúdo
  22. cp - copia o conteúdo de um arquivo em outro, obs.: para cp copiar um diretório, com conteúdo, é preciso utilizar o argumento -r
  23. mv - move o arquivo ex.: mv arq.txt novoDiretorio

Editando texto no terminal

  1. vi nomeDoArquivo.txt - permite editar o arquivo direto do terminal
  2. tecla i para inserir texto na posição atual;
  3. tecla a para inserir texto no caracter seguinte;
  4. tecla A para inserir texto no final da linha;
  5. tecla x para remover um caracter, pode receber um número antes do x para deletar mais de um caractar por vez;
  6. tecla Esc para sair do modo de edição;
  7. tecla G move o cursor para a última linha;
  8. teclas 0G1G move o cursor para a primeira linha;
  9. teclas 7G move o cursor para a sétima linha (7 representa qualquer número);
  10. teclas dd para remover uma linha inteira;
  11. teclas yy para copiar uma linha inteira;
  12. teclas 9yy para copiar nove linhas (9 representa qualquer número);
  13. tecla p para colar;
  14. teclas 4p para colar quatro vezes o mesmo conteúdo (4 representa qualquer número);
  15. >utilizar as teclas de seta para navegar pelo documento;
  16. tecla : para habilitar linha de comando;
  17. comando w para salvar (w=write);
  18. comando q para sair (q=quit);
  19. comando q! para sair sem salvar;
  20. comando wq para sair e salvar;
  21. tecla / nomeDaPalvra realiza uma busca por nomeDaPalvra (n vai para a próxima ocorrencia; N volta para a ocorrencia anterior);

Ferramentas de Compactação

  1. zip nomeArq.zip nomeDir - compacta o diretório nomeDir, criando um arquivo nomeArq.zip
  2. zip -r nomeArq.zip nomeDir - utiliza-se o -r para compactar tudo que está dentro do diretório
  3. zip -rq nomeArq.zip nomeDir - o argumento q resume as informações na tela
  4. unzip - descompacta os arquivos
  5. unzip -l - lista o conteúdo do arquivo
  6. tar -czf nomeDoArquivo.tar.gz nomeDoDiretorio - compacta o diretorio gerando um arquivo .tar.gz

    7. c - cria; z - zipa; f - file, ou seja, nome do arquivo

  7. tar -xzf nomeDoArquivo.tar.gz - para descompactar
  8. tar -vxzf nomeDoArquivo.tar.gz - o argumento v, mostra detalhes do processo
  9. tar -cjf nomeDoArquivo.tar.bz2 nomeDoDiretorio - forma mais eficiente de compactar, pode levar mais tempo
